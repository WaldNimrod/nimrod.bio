# LOD400 — NB-S001-P002-WP001 — Environment Bootstrap

## 1. מטרה

הקמת סביבת פיתוח מקומית ל-nimrod.bio שמשקפת את production ב-uPress.
ה-stack המקומי יאפשר עבודה על תבנית, עיצוב ותוכן ללא מגע בשרת החי.

## 2. תנאי קדם (Team 00 אחראי)

לפני שהסקריפטים רצים, Team 00 מניח בתיקייה `sources/`:
- `*.zip` — full-site backup מ-uPress (WordPress files + wp-content)
- `*.sql` — MySQL dump של ה-DB מ-uPress

**חשוב:** קבצים אלו אינם מוחייבים ל-git (`.gitignore` מגדיר זאת).

## 3. מה לבנות

### 3.1 מבנה תיקיות

```
nimrod-bio/
├── nimrod.bio/          ← site root תחת git (wp-content/, wp-config-sample.php)
├── sources/             ← ZIP + SQL (gitignored)
├── docker-compose.yml
├── .gitignore
└── scripts/
    ├── restore-production-from-backup.sh
    └── verify-connections.sh
```

### 3.2 docker-compose.yml

שירותים:
- **wordpress** — image: `wordpress:latest` (או `wordpress:php8.2`)
  - Port: `8085:80` ← **חייב להיות 8085** (registered port, Iron Rule #9)
  - Volumes: `./nimrod.bio:/var/www/html`
  - Environment: `WORDPRESS_DB_HOST`, `WORDPRESS_DB_NAME`, `WORDPRESS_DB_USER`, `WORDPRESS_DB_PASSWORD`
- **db** — image: `mysql:8`
  - Port: `3309:3306` ← **חייב להיות 3309** (registered port)
  - Environment: `MYSQL_ROOT_PASSWORD`, `MYSQL_DATABASE`, `MYSQL_USER`, `MYSQL_PASSWORD`
  - Volume: mysql_data

### 3.3 scripts/restore-production-from-backup.sh

הסקריפט:
1. מאתר `sources/*.zip` — מחלץ לתיקייה זמנית
2. מעתיק את תוכן ה-zip ל-`nimrod.bio/` (overlaying)
3. מאתר `sources/*.sql` — מייבא ל-MySQL (`docker exec` + `mysql`)
4. מריץ search-replace ב-DB: URL ישן (https://nimrod.bio) → `http://localhost:8085`
   - שיטה: WP-CLI בתוך container, או Python script עם regex ב-SQL dump לפני import
5. מחייב הודעה: `"Restore complete: http://localhost:8085"`

**דרישות בטיחות:**
- לא לכתוב credentials ב-script — לשאוב מ-`.env.local` (gitignored) או Docker env vars
- לא לבצע `git add sources/` בשום שלב
